Skill 說明

訓練 Rocky 去使用 skill 做事情可以讓事情穩定的完成，尤其是那些需要重複性、或是要標準化的工作。

skill 訓練方式：

  1. 呼喚 Rocky ，請他做一件事情
  2. 完成後，請他把過程變成一個 skill 記錄起來
  3. 請他再用一次剛剛的 skill 做同樣事情，確認結果相同
  4. 有錯誤的話請他修正
  5. 重複以上步驟，直到你覺得他這個 skill 穩定為止

skill 訓練技巧：

  1. 一個 skill 只做一件事情
  2. 長的流程盡量切成多個小步驟，每個步驟一個 skill
